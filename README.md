# API Flask - Exemplo de Autenticação com Token Bearer

Descrição

Esta é uma API desenvolvida em Flask para demonstrar conceitos básicos de desenvolvimento de APIs REST.

O projeto apresenta:

- Rotas GET e POST
- Recebimento de parâmetros pela URL
- Recebimento de JSON no corpo da requisição
- Uso de Headers HTTP
- Autenticação utilizando Bearer Token
- Retorno de códigos HTTP (200, 400 e 401)

## Tecnologias utilizadas

- Python 
- Flask
- Requests (para testes)
- python-dotenv (para variáveis de ambiente)
